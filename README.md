# Coffee Shop Sales Analysis (Excel Project)

## Project Overview
This project analyzes coffee shop transaction data to understand sales trends,
payment behavior, and product performance using Microsoft Excel.

## Objectives
- Analyze total sales over time
- Compare cash vs card payments
- Identify top-selling coffee products
- Build a KPI dashboard for decision-making

## Tools Used
- Microsoft Excel

## Dataset
- 3,636 transactions
- Payment types: Cash & Card
- Products: Multiple coffee types
- Data anonymized for analysis

## Key Insights
- Card payments accounted for most transactions
- Certain coffee types consistently generated higher revenue
- Sales peaked during specific hours
